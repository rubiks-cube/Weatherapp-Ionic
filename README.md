

# Weatherapp-Ionic  
A simple hybrid weather app using Ionic framework.  
## Requirements:
-Ionic  
-AccuWeather API   
-Angular-maps   
## Features:  
-User can get the current weather conditions of:   
  (i) his current location   
  (ii) or any place in the world by searching.
